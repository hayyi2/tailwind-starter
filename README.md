## Tailwind Starter
Simple Template Starter with Tailwind, Nunjucks, Browsersync, and Gulp.js.

## 🚀 Getting Started

1. Clone repo
    
    `git clone https://github.com/hayyi2/tailwind-starter.git`
2. Install packages
   
   `npm install`
3. Run Project
   
   `npm start`

   to build
   
   `npm run build`


Have fun 🎉🎉
